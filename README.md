# 划词翻译助手 - Chrome Extension

选中网页文字，即刻翻译为中文、英文、法语，支持自然朗读与逐词朗读。

## 功能

- **三语互译** — 自动识别中/英/法，翻译为另外两种语言
- **自然朗读** — 整句流畅播放，听真实发音
- **逐词朗读** — 逐词播放 + 蓝色高亮同步，适合学习发音
- **语音选择** — 在插件弹窗中自定义每种语言的朗读声音，支持试听
- **免费使用** — 基于 MyMemory 翻译 API，无需注册

## 安装

1. 下载或克隆本仓库
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角打开 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本项目文件夹

## 使用

1. 在任意网页中选中文字
2. 翻译浮层自动弹出，显示三语翻译结果
3. 每行有两个按钮：
   - 左侧喇叭 — 自然朗读（整句流畅）
   - 右侧图标 — 逐词朗读（高亮同步）
4. 点击插件图标可设置每种语言的朗读声音

## 项目结构

```
├── manifest.json    # Chrome 扩展配置 (Manifest V3)
├── content.js       # 核心逻辑：划词检测、翻译、朗读
├── content.css      # 翻译浮层样式
├── popup.html       # 插件弹窗 UI
├── popup.js         # 语音设置逻辑
└── icons/           # 插件图标
```

## 技术栈

- Chrome Extension Manifest V3
- Web Speech API (语音合成)
- MyMemory Translation API (免费翻译)
